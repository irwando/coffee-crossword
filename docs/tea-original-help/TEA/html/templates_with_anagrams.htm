Templates with Anagrams

The concepts of Templates and Anagrams can be combined in a single pattern. You use a semi-colon (;) to indicate where the anagram starts. TEA only displays the words that match both the template and the anagram parts of the pattern.

This type of search is useful for solving anagram clues where you know some of the letters from other answers already entered into a crossword grid.

It is also useful for finding the longest words from a set of letters, such as those found on a Scrabble® rack or in Countdown. Specifying a template that consists of a number of Match-All Characters and a Wildcard ensures that very short words are not displayed; you could also use the Shortest setting on the TEA Main Window to do the same thing.

Anagram Balances are particularly useful with this kind of pattern: unused letters in the anagram are shown after a minus (-); if a dot is included in the anagram to represent a blank tile in Scrabble®, the letter used for the blank is shown after a plus (+).
