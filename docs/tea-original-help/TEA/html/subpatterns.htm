Sub-Patterns

If you only want to represent part of a word using a pattern, you can do this with sub-patterns. You indicate a sub-pattern by enclosing it in round brackets ().

Typical uses of sub-patterns are to:

  • include an Anagram within a Template; for example, if part of a crossword answer is known to be an anagram of a set of letters, this can be represented with a sub-pattern
  • include a Template within an Anagram; for example, if a crossword answer is known to be a jumble of a fixed sequence of letters and other letters in random order, this can be represented with a sub-pattern

Examples

...(;orange) finds nine-letter words where the last
six letters are an anagram of ORANGE		 
patronage
;rebel(ada) finds words with ADA in sequence and
in order;
and the letters of REBEL in any order		 
readable
;umber(;lily) finds words with LILY in sequence, but
not necessarily in order;
and the letters of UMBER in any order
