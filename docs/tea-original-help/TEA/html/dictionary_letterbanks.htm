Letter Banks Dictionary Licensed Version Only

The Letter Banks Dictionary includes English words and phrases that make a "letter bank", in which all the letters of a "bank" word are used (as many times as needed, but at least once) to make a new word at least three letters longer.

This dictionary can be used to create or solve puzzles where the answer is known to have such a constraint. Because the Letter Banks dictionary has such a specialized purpose, it is not opened by default, but can be opened using the File|Open Command.

The dictionary includes two types of entries:

  • the banks (tagged with a $ at the end), each having an Annotation to show the longer words made from them
  • longer words, each having an Annotation showing from which banks they can be derived

For example:

Entry Annotation
imps$ Mississippi
Tennessee nest$,nets$,sent$,sten$,tens$
amuse$ masseuses,masseuse
