Options Pattern Tab

Use the Options Pattern Tab to change the way Macros and Letter Variables are treated in patterns. To access these settings, use the Tools|Options Command and click on the Pattern tab.

Click on Save as defaults to use the settings on the tab as the defaults when TEA starts up.

@ becomes
# becomes

The macro characters @ and # in patterns are replaced by the text you enter here.

The default setting for @ is [aeiou] and the default setting for # is [^aeiou]. This means they can be used to represent a vowel in a word and a consonant in a word.

Exclusive letter variables

There are two ways Letter Variables can be interpreted in patterns:

  • with exclusive letter variables, each number must equate to a different letter
  • with non-exclusive letter variables, different numbers can equate to the same letter

If you are trying to solve Cryptograms, exclusive letter variables tend to be more appropriate. If you are searching for Palindromes and the like, you need non-exclusive letter variables. For example:

122..33. with exclusive letter variables finds 
annually
root-beer
with non-exclusive variables also finds 
cook-book
offshoot
pooh-pooh
123454321 with exclusive letter variables finds 
Rotavator
with non-exclusive variables also finds 
Malayalam

Ignore unused punctuation

In Punctuation Matching, there are two ways TEA can deal with punctuation characters (eg space, hyphen or apostrophe) that aren't used anywhere in the pattern:

  • if you check Ignore unused punctuation, TEA will only match whatever punctuation characters you include in the pattern (and those must be matched exactly)
  • if you uncheck Ignore unused punctuation, TEA will match all punctuation characters (and all must be matched exactly)

For crossword solving, it is usually best to keep Ignore unused punctuation checked: that's because the Enumeration may tell you where some of the punctuation characters occur, but not all. For example, you may be able to tell where there are spaces, and sometimes hyphens, but not apostrophes.

Here are some example patterns where this option makes a difference:

..d..-....l. with Ignore unused punctuation checked finds
(hyphens are significant, but apostrophes are ignored)		 
Baden-Powell
lady's-mantle
with Ignore unused punctuation unchecked finds just
(all punctuation characters must be matched exactly)
