TEA Quick Reference Window

The TEA Quick Reference window is a reminder of the different types of pattern. It is particularly helpful when you are learning to use TEA or starting to use unfamiliar types of pattern. To open the Quick Reference, use the Help|Quick Reference Command.

You can get further help by clicking on the underlined links. Click on a pattern to search for it using your current configuration.

Show this window when TEA starts

If this setting is checked, the TEA Quick Reference window appears by default when TEA starts. Uncheck this if you don’t normally want to view the Quick Reference.
