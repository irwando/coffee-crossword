Edit|Lookup Command

Use the Edit|Lookup command (which has Ctrl+Enter as a shortcut) to view the Dictionary Definitions for the selected word in a TEA Definition Window.

By default, double-clicking on an entry in a results list is another shortcut for Edit|Lookup (however, this can be changed using the Double-click action setting on the Options Result Tab).

Edit|Lookup goes through the following sequence to find definitions:

  1. it searches each Dictionary open in the main window
  2. it searches the Definition Dictionaries configured on the Options Dictionary Tab
  3. if no definitions are found from either of the above steps, it opens the External Lookup Message Box so you can try an external lookup instead
