Main Window Results List

Dictionary results lists show the entries in the Dictionary that were matched during a search. If the results do not fit in the space available, a scrollbar appears allowing you to page through the results. If you position the mouse pointer over a partially visible entry, the complete entry is shown in a tooltip.

You can use the mouse or keyboard to select entries to be processed. Clicking the right mouse button over an entry will select it and pop up the Results Context Menu.
