Logical Operations

You can use Logical Operations to combine the results from different patterns together. This is particularly useful when the Search Mode is Full Text Search, as it helps you to cut down the number of Dictionary Definitions that are matched.

TEA allows three commonly used logical operations in patterns: AND (represented by & in patterns), OR (represented by |) and NOT (represented by !).

The following examples show logical operations used in Word Search mode. For examples in Full Text Search mode, see Searching for Definitions.

Examples

1*1 & *date* finds words that start and end with the same letter AND include the word DATE somewhere 
dated
elucidate
sedateness
dilapidated
....cous | cous.... finds eight-letter words that either start or end COUS; note that couscous, which does both, is only matched once 
aduncous
caducous
couscous
cousinly
cousinry
glaucous
muticous
;tearing & ! *ing* finds anagrams of TEARING that do not contain the letters ING in sequence 
granite
gratiné
(p* | *p) & ;q...... finds seven-letter words starting or ending with P and containing Q 
parquet
piquant
prequel
reequip
