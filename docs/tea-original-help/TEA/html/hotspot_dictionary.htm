Main Window Dictionary Header

A dictionary header shows the name of the Dictionary used to produce the results below it. If you position the mouse pointer over a header, the description of the dictionary appears in a tooltip.

The underlined number to the left of the dictionary name indicates the keyboard short cut that can be used to select the dictionary. The number in brackets to the right of the dictionary name indicates how many matching entries were found during the search.

The color of the dictionary header indicates whether the dictionary is active; menu commands relating to a dictionary or its search results only affect the active dictionary.

Clicking the right mouse button over a dictionary header selects the dictionary and pops up the Dictionary Context Menu.
