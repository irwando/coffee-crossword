Searching for Definitions

You can search for Dictionary Definitions that contain one or more words within them. You do this by changing the Search Mode to Full Text Search. The normal types of patterns are used in this mode, but instead of searching for words you are searching for words within definitions.

Changing to Full Text Search mode

The normal search mode is Word Search. To change the Search Mode to Full Text Search, click on the down arrow to the right of the Word Search button and use the Full Text Search Command.

If you often change between Word Search and Full Text Search modes, you may find it more convenient to use the keyboard shortcuts Ctrl+W (for Word Search) and Ctrl+F (for Full Text Search).

Keyboard users can also change the Search Mode by using the Tab key to set the focus to the Word Search button and pressing the APPLICATION Key which pops up the Search Context Menu.

Using Full Text Search mode

In Full Text Search mode, you enter patterns in the normal way (see Overview of Patterns), but the patterns will be used to match words within definitions.

The other important difference is in the way Logical Operations apply. In Full Text Search mode, AND, OR and NOT relate to the existence of any word that matches the patterns they connect, not necessarily the same word. The examples below will illustrate this.
