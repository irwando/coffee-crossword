Page Setup Dialog

Use the Page Setup dialog to select the paper size, orientation and margins used for printing. To open the dialog use the File|Page Setup Command.

Paper

Select the size of paper you will be using from those supported by your printer.

Orientation

Select whether the paper will be printed in Portrait or Landscape mode.

Margins

Printed results will appear within the margins you set in this section.

Printer…

Click on this button to select the printer and change its settings.
