TEA Tip Window

The TEA Tip window gives brief information about a single TEA feature. It helps you learn to use TEA a little bit at a time. To open the Startup Tips, use the Help|Startup Tips Command.

You can get further help by clicking on the More on this topic button. Click on the Next button to see the next tip in the loop.

Show this window when TEA starts

If this setting is checked, the TEA Tip window appears by default when TEA starts. Uncheck this if you don’t normally want to view a startup tip.
