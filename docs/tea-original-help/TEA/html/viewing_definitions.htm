Viewing Definitions

There are several ways you can open a TEA Definition Window to view the Dictionary Definitions for a word:

From a Results List

Select a word in a results list in the TEA Main Window and use the Edit|Lookup Command (which has Ctrl+Enter as a shortcut) or Edit|Lookup in New Window Command (which has Shift+Ctrl+Enter as a shortcut).

Double-clicking on an entry in a results list is a quick way of doing the Edit|Lookup Command.

From Another Definition

Select a word in a definition in a TEA Definition Window and click on the Lookup button in the toolbar. If you hold the Shift key down while clicking on the Lookup button, the definition will appear in a new window.

Double-clicking on a word in a definition is a quick way of doing lookup. If the word is part of an underlined phrase, the complete phrase will be looked up.

The Definition Context Menu offers an alternative way of doing lookup that keyboard users may find helpful.

Using the Pattern Entry Box

Enter a word in the Pattern entry box in the TEA Main Window and use the Edit|Lookup Command (which has Ctrl+Enter as a shortcut) or Edit|Lookup in New Window Command (which has Shift+Ctrl+Enter as a shortcut).
