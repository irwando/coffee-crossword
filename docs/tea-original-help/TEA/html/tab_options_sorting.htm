Options Sorting Tab

Use the Options Sorting Tab to change the way results lists are sorted. To access these settings, use the Tools|Options Command and click on the Sorting tab.

Click on Save as defaults to use the settings on the tab as the defaults when TEA starts up.

Sorting

Check Sort Alphabetically to see the results in alphabetical order. If Group by length is also checked, the results are separately sorted for each word length.

If Ascending is unchecked, the results are reverse alphabetical order.

Length grouping

Check Group by length to see a separate group of results for each word length.

You control the order in which different lengths of answers are shown with these settings:

  • Ascending: shortest answers at the top, longest answers at the bottom
  • Smart: shortest Anagram Balances at the top, longest Anagram Balances at the bottom
  • Descending: longest answers at the top, shortest answers at the bottom
