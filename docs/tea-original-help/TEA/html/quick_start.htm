Quick Start

If you are an experienced Windows user, this one page summary should be enough to get you started.

Type a search pattern in the box labeled Pattern, then press the return key (or click the Word Search button) to see the matched words for each Dictionary in the lists below.

The most commonly used pattern is a Template, in which the known letters are separated by dots (.) or question marks (?). To search for Anagrams of a set of letters, precede the letters with a semi-colon. For example:

Pattern Type Example Matches
Template 
.l...r.n
 
electron
flatiron
glycerin
Anagram 
;acenrt
 
canter
nectar
recant
trance

Those are the two basic patterns. The TEA Quick Reference shows a reminder of the pattern features, with examples to try. You can combine the different types of pattern in hundreds of ways to create the exact search you want. To see the possibilities, browse through the pattern help pages starting with the Overview of Patterns or try the TEA Online Tutorial.

To look up the Dictionary Definitions for a matched word, double-click on it, or select it and use the Edit|Lookup Command. The definitions are shown in the TEA Definition Window.
