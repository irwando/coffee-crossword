Pattern Processing

You can customize these aspects of TEA Patterns:

Lengths of Words

It is sometimes helpful to limit the range of word lengths that are matched by a pattern. You can do this using the Shortest and Longest settings in the TEA Main Window.

Treatment of Letter Variables

You can indicate that different Letter Variables must always match different letters of the alphabet using the Exclusive letter variables setting on the Options Pattern Tab.

Expansion of Macros

You can change what the Macros expand to using the @ becomes and # becomes settings on the Options Pattern Tab.
