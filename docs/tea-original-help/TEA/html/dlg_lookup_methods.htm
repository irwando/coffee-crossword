Lookup Methods Dialog

Use the Lookup Methods dialog to configure the Edit|External Lookup Commands and identify which is the default one.

If the box at the left of a method is checked, the method is enabled and will appear in the External Lookup Submenu. The single method with a black check mark is the default (the one invoked when a shortcut such as Ctrl+Alt+Enter is used).

The method's Name becomes the name of the command in the External Lookup Submenu. If you precede a letter of the name with an ampersand (&) that will be underlined in the menu, indicating that the letter acts as a shortcut.

For an up-to-date list of Internet search engines and applications you can configure, see the Dictionary Software page of the Crossword Man web site.

Web Searches

Internet search engines are the most commonly used external lookup methods. Enter the address used to do the search in the box to the right of the Address: label. Include {entry} to indicate where the word or phrase being looked up should appear as part of the address.

To switch off a lookup method, click the box at the far left till the check mark disappears. To make a lookup method the default, click the box till a black check mark appears (the check marks for the other enabled methods will turn gray indicating that they are not the default method).

Legacy Application Support

TEA also supports some older Electronic Dictionary applications that don't use the standard clipboard copy interface to identify words to be looked up. Three styles of interface are possible:

  • DDE Interface. Some Electronic Dictionary applications (such as those based on the CompLex software) support lookup from other applications using Dynamic Data Exchange.
  • Message Interface. With some other Electronic Dictionary applications (such as the American Heritage Dictionary for Windows), applications must send a Windows message to look up a word.
  • Application Interface. Some Electronic Dictionary applications (such as the Cassell Concise Dictionary) include an application that, when run, looks up a word.

All these methods are switched off by default. If you have one of the supported applications, you can switch on its entry by checking the box at the far left of the dictionary row.

Test word

You can test the default method (the one with a black check mark) by entering the text to be looked up in this box and clicking on the Test Lookup button. If the application uses a DDE or Message Interface, you need to start it before running the test.

Save as defaults

Click on this button to use the settings as the defaults when TEA starts. Otherwise they will only apply for the current TEA session.
