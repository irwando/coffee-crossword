Cryptogram

A puzzle in which a message is encoded by representing each letter of the alphabet with another at random. Solvers have to use the word breaks and patters of letters to decode the message. A short example of a cryptogram is: 

CLX GNXNMMNP LTZ CLX EIFGIMZLE, WDG TIUIM CLX GNZLE

The solution is “jam tomorrow and jam yesterday, but never jam today”.
