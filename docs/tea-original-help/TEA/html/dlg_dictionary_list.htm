Dictionary List Dialog

Use the Dictionary List dialog to add and remove Dictionaries in a Dictionary List and to change their order.

The list labeled Selected dictionaries shows the Dictionaries currently in the Dictionary List, with the one that is used first at the top. The list labeled Other dictionaries shows the other available Dictionaries in the selected folder.

To move a Dictionary in the filling order, select it in the Selected dictionaries list and click on the Promote or Demote buttons. To stop using a Dictionary, select it and click on the Remove button. To use a new Dictionary, select it in the Other dictionaries list and click on Add.

Information about each Dictionary is shown in a tooltip when you position the mouse over the dictionary name. For complete information about a Dictionary, select it and click on the Properties… button.

If there are many Dictionaries, they may be grouped under different folders. Use the Browse… button to choose which folder is listed in the dialog.
