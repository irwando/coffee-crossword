Navigating between Definitions

There are several ways to navigate to related definitions using the TEA Definition Window. These are modeled on the navigation commands for popular web browsers.

You can find all the navigation commands on the Definition Context Menu, although you rarely need to use the menu as the toolbar includes buttons for the most commonly used commands.

Navigating to the Definitions for Another Word

The quickest way of looking up the Dictionary Definitions for a word is to double-click on it (you can do this whether the word is underlined or not). TEA will assume that you are interested in the complete word indicated; if the word is part of an underlined phrase, it will assume you are interested in the complete phrase.

If you want to look up part of a word, select the text you are interested in and click the Lookup button in the middle of the toolbar. This is a quick way of doing the Lookup Command, which has Ctrl+Enter as a keyboard shortcut.

To look up a word on the Internet (or in a supported external Electronic Dictionary application), use the External Lookup button. This is a quick way of doing the default External Lookup Command which has Ctrl+Alt+Enter as a keyboard shortcut.

Navigating within the Results List

TEA users often want to view the Dictionary Definitions for a series of words in the results list. You can navigate up and down the results list using the GoTo|Next Result Command and GoTo|Previous Result Command.

The Next Result and Previous Result buttons at the left hand side of the toolbar are a quick way of doing these commands. Their keyboard shortcuts are Alt+Down Arrow and Alt+Up Arrow.

Navigating within the History List

Another frequent requirement is to go back to previously viewed Dictionary Definitions and then forward to ones viewed later. You can navigate through the history list using the GoTo|Back Command and GoTo|Forward Command.

The Back and Forward buttons at the right hand side of the toolbar are a quick way of doing these commands. Their keyboard shortcuts are Backspace (or Alt+Left Arrow) and Alt+Right Arrow.

To see a list of all the recently viewed words, click with the right mouse button and select the Go To submenu. This lists the last ten words viewed at the bottom and you can revisit any of these by selecting the corresponding menu item.
