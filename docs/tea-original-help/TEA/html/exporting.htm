Exporting

If you want to use the results of a search using another Windows application you can either copy them to the clipboard or save them to a file.

Clipboard Copy

Use the Copy Command: you can access this by clicking with the right mouse button on the results list you want to export – this pops up the Result Context Menu; this command is also on the Edit Menu.

Clipboard copy only exports the selected results (see Selection). If you wish to export all the results, use the Select All Command first.

Saving to File

Use the Save Results As Command: you can access this by clicking with the right mouse button on the header of the results list you want to export – this pops up the Dictionary Context Menu; or by using the File Menu (the name of the command indicates which results list will be saved).

You can save the results as a text file or as a TEA&Sympathy Dictionary File. You use the Save Results Dialog to choose the type of file and its name.

Note that if you create a new TEA&Sympathy Dictionary File in this way, it will only contain the words in the results list, not their Dictionary Definitions or Annotations.
