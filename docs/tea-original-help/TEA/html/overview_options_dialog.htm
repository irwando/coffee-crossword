Options Dialog Overview

Use the Options dialog to customize the way the TEA Main Window works.

Click on the tabs at the top of the dialog to select different types of options. You can find out more about each tab using the links below.

The changes you make apply to the current use of TEA, unless you click on the Save as defaults button to make the settings on the tab the permanent defaults.

Pattern Tab Treatment of Macros and Letter Variables in patterns
Result Tab Presentation and filtering of results lists
Sorting Tab Sorting of results lists by length and/or alphabetically
Dictionary Tab Treatment of multiple Dictionary searches
Appearance Tab Fonts used for display and printing

You can keep the Options dialog open while you continue to work in the TEA Main Window. Keyboard users may find the following function keys helpful for navigating between the dialogs and the main window:

F4 switches to the main window and selects the Pattern box
F5 switches between the dialog window and the main window
F6 cycles through all the open dialogs
