Macros

A Macro is an abbreviation (represented by either @ or #) for a commonly used part of a pattern. By default, @ expands to [aeiou] (a Choice List indicating a vowel) and # expands to [^aeiou] (a Choice List indicating a non-vowel).

You can customize how these macros are expanded using the Options Pattern Tab, so they can be used for a great variety of purposes. The examples below show how the default macro settings can be used to investigate words with unusual arrangements of vowels and consonants.

Examples

;@@@@# finds words with one consonant
and four vowels		 
adieu
aerie
audio
eerie
queue
##@@##@@## finds words with vowels and consonants
in pairs		 
bloodhound
floorboard
sweetheart
