Print Preview Window

The Print Preview window shows what the printed TEA results will look like. To open the window, use the File|Print Preview Command.

The caption at the top of the window shows the number of the displayed page. Resizing the window allows you to see the page at different zoom factors. The buttons represent the following commands:

Show all the results in the Print Preview window

Show only the selected results in the Print Preview window

Select the previous page (you can use the left arrow key as a shortcut for this)

Select the next page (you can use the right arrow key as a shortcut for this)

Print the selected page

Use the Options Appearance Tab to change the font used for printed output. Use the Page Setup Dialog to change the margins, paper size and orientation.
