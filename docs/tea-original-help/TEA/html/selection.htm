Selection

Many results list features can be applied to a selection of entries in the list. To select all the entries, use the Edit|Select All Command. You can select a subset using the mouse or the keyboard:

Mouse Selection

To select a single entry If necessary, use the scroll bar to scroll the item into view, then click on it with the left mouse button.
To select multiple sequential entries Press the left mouse button down over the first item to be selected, then drag the cursor to the last item and release the mouse button.

Alternatively, you can select the first item; then press and hold down the shift key; then click on the last item and release the shift key.
Selecting non-sequential entries Press and hold down the Ctrl key; then click with the left mouse button to select an entry; clicking on a selected entry will deselect it.

To select or deselect multiple sequential entries, drag with the mouse while the Ctrl key is held down.

Keyboard Selection

To select a single entry Use the arrow keys until the required entry is selected.

You can use the Home and End keys to move the selection to the top and bottom. You can use the PgUp and PgDn keys to move the selection up and down a page.
To select multiple sequential entries Use the arrow keys until the first entry is selected; then press and hold down the shift key; use the arrow keys until the last entry is selected and release the shift key.

You can use the Home and End keys to extend the selection to the top and bottom. You can use the PgUp and PgDn keys to extend the selection by a page at a time.
To deselect a single entry Press and hold down the Ctrl key, then press the space bar.
