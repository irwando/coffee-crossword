Anagrams

To search for an Anagram of a set of letters, start with a semi-colon (;). You can use dot (.) or question mark (?) as Match-All Characters to indicate unknown letters – the letters used to replace the unknowns are shown after a + in Anagram Balances.

This feature helps you work out long anagrams such as those on Countdown and solve anagram clues, even if you don’t know all the letters of the anagram.

Examples

;acenrt shows all the words containing
A, C, E, N, R and T		 
canter
nectar
recant
trance
;oldmaneli finds 
medallion
;theodora.. finds (using two different
replacements for the dots)		 
fatherhood +FH
parenthood +NP
