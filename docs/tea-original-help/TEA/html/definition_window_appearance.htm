Definition Window Appearance

You can customize TEA Definition Windows in the following ways:

Window Size

You can change the size of a TEA Definition Window by dragging the window at the edges or using the sizing grip in the bottom right hand corner.

To make the window occupy the whole screen, use the Maximize button in the title bar.

Fonts

You can change the default font and text colors for definitions using the Definition font settings in the Options Appearance Tab. The word or words being looked up are highlighted in a different font and color which you can change using the Definition highlight font setting.

Note that when you build a TEA&Sympathy Dictionary File you can specify that a particular font is used for all or part of a definition. In such cases, the Definition font setting will have no effect, since the font specified by the dictionary designer takes precedence.
