Results Presentation

You can customize the way Dictionary results appear:

Multiple Dictionary Handling

By default, each matched result is only shown for the highest priority dictionary and suppressed in the remaining dictionaries. You can also opt to see all the results for all the dictionaries, or only search the selected dictionary, using the Options Dictionary Tab.

Filtering Types of Answer

If you know that the answer you are looking for is (or is not) a phrase, hyphenated word or proper noun, you can suppress inappropriate types of answer using the Filtering settings on the Options Result Tab. By default, all types are shown.

Anagram Balances

By default, Anagram Balances are shown at the end of results for anagram searches. You can suppress these by unchecking the Show anagram balances setting on the Options Result Tab.

Anagram balances are shown using the same special font as for Annotations. This font is changed using the Results Annotation Font setting in the Options Appearance Tab.

Sorting

By default, results are shown in increasing order of length; and for each length, in the order that was present when the dictionary was built using the TEA&Sympathy Dictionary Builder.

If you want to see the results alphabetically, regardless of length, check the Sort alphabetically setting on the Options Result Tab.
