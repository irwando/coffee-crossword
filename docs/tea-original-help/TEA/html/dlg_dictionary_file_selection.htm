Dictionary File Selection Dialog

Use the Dictionary File Selection dialog to open one or more TEA&Sympathy Dictionary Files for searching. To open the dialog, use the File|Open Command.

To open a single dictionary file, select it with the left mouse button and click on the Open button.

To open multiple dictionary files in the same folder, press and hold down the Ctrl key; then click on the dictionary files to select them (clicking again on a selected dictionary deselects it); then release the Ctrl key and click on the Open button.

You can also use the Dictionary File Selection dialog to manage your dictionary files. For example, you can rename or delete a dictionary file using the menu that appears when you click on it with the right mouse button. You can create new folders and move dictionary files to them using drag and drop. This functionality is common to all Windows applications and is described in more detail in the Windows help file.
