Save Results Dialog

Use this dialog to choose the name and type of results file to create. To open this dialog, use the File|Save Results As Command. Enter the name of the file that you want to create in the File name box and choose the file type:

  • Text Document (TXT) – creates a text file containing the results;
  • TEA&Sympathy Dictionary (TSD) – creates a TEA&Sympathy Dictionary File containing the results (it is acceptable to choose a dictionary already in use, as TEA doesn't remove the previous dictionary until the new one has been created successfully). When you click on Save, the Save TEA&Sympathy Dictionary Dialog will appear, allowing you to add a description before saving begins.
