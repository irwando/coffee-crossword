Dictionary Selection and Ordering

The ability to search multiple Dictionary files is one of TEA’s most powerful features. If you prioritize the dictionaries in the right way, you can find the required word more quickly: it’s usually best to search the smallest dictionary first and the largest dictionary last.

Obtaining Dictionaries

TEA searches four English language word lists with over 5 million unique entries as standard (see Overview of Dictionaries). Another 23 dictionaries are provided in the TEA Dictionaries subfolder and can optionally be opened for searching. If you need additional TEA&Sympathy Dictionary Files you can incorporate your own word lists using the TEA&Sympathy Dictionary Builder.

Opening Dictionaries

There are several ways to select the dictionaries that TEA searches:

  • use the File|Open Command to select one or more additional dictionaries for searching
  • use the File|1,2,3,4,5,6,7,8,9,0 Commands to select a previously used dictionary for searching
  • drag one or more TEA&Sympathy Dictionary Files from Windows Explorer and drop them on the TEA Main Window
  • double-click on a TEA&Sympathy Dictionary File in Windows Explorer to start TEA with that dictionary selected for searching
  • drag one or more TEA&Sympathy Dictionary Files from Windows Explorer and drop them onto the TEA Application icon to start TEA with the set of dictionaries selected for searching

Prioritizing Dictionaries

You can raise or lower a dictionary in the priority order:

  • use the File|Promote Command to raise the dictionary in the search order
  • use the File|Demote Command to lower the dictionary in the search order

Closing Dictionaries

To stop searching a dictionary, use the File|Close Command.
