Main Window Appearance

You can customize the TEA Main Window in the following ways:

Window Size

You can change the size of the TEA Main Window by dragging the window at the edges or using the sizing grip in the bottom right hand corner.

To make the window occupy the whole screen, use the Maximize button in the title bar.

Multiple Dictionary Layout

You can adjust the space allotted to each results list by dragging the borders between them. You can choose between a horizontal or vertical arrangement of results lists:

  • use the Window|Tile Horizontal Command to place the results lists one on top of the other; this arrangement is best if you use a narrow window
  • use the Window|Tile Vertical Command to place the results lists side by side; this arrangement is best if you use a full-size window

Fonts

You can change the font and text colors used for the pattern entry and the results lists using the Pattern and Results settings in the Options Appearance Tab.

It is usually best to have a fixed width font such as Courier for the Pattern box as this makes it easier to enter long Templates. A fixed width font can also help in results lists, as words with the same length then line up.
