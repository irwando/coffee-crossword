Options Appearance Tab

Use the Options Appearance Tab to change the fonts and colors used in TEA. To access these settings, use the Tools|Options Command and click on the Appearance tab.

Click on Save as defaults to use the settings on the tab as the defaults when TEA starts up.

The first three settings affect the TEA Main Window. Click on the buttons to the right of the labels to change the font and text color indicated:

Setting Determines
Pattern font The font and text color in the pattern entry box
Results font The font and text color in results lists
Results annotation font The font and text color used for Anagram Balances and Annotations
Printout font The font and text color in printed results

The remaining settings affect TEA Definition Windows. Click on the buttons to the right of the labels to change the font and color indicated:

Setting Determines
Definition background color The definition window background color
Definition font The default font and text color used in definition windows
Definition highlight font The font and text color used to highlight the lookup word in definition windows
