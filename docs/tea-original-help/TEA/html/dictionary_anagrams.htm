Anagrams Dictionary Licensed Version Only

The Anagrams Dictionary includes English words and phrases that are an Anagram of another word (or phrase) – words and phrases without Anagrams are excluded.

This dictionary can be used to solve puzzles where the answer is known to be an Anagram of something else. Because the Anagrams dictionary has such a specialized purpose, it is not opened by default, but can be opened using the File|Open Command.

Each entry includes an Annotation that shows its possible Anagrams. For example:

Entry Annotation
ratio ariot
recentness secernents,sentencers
