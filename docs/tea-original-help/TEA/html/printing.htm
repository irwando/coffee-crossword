Printing

You can print all or some of the results of a Dictionary search and have the option of printing the whole selection immediately, or printing the results page-by-page using a preview.

Immediate Printing

Use the Print Results Command: you can access this by clicking with the right mouse button on the results list you want to print – this pops up the Result Context Menu; or by using the File Menu (the name of the command indicates which results list will be printed).

If you selected any results (see Selection) you then have the option to print All the results or just the Selection using the buttons in the Print range box of the Print Dialog.

Printing with Preview

Use the Print Preview Results Command: you can access this by clicking with the right mouse button on the results list you want to preview – this pops up the Result Context Menu; or by using the File Menu (the name of the command indicates which results list will be previewed).

The pages are viewed in the Print Preview Window.
