Options Dictionary Tab

Use the Options Dictionary Tab to change the way multiple dictionary searches are handled and choose which Dictionaries are used by the Edit|Lookup Command. To access these settings, use the Tools|Options Command and click on the Dictionary tab.

Click on Save as defaults to use the settings on the tab as the defaults when TEA starts up.

Multiple Dictionary Searching

You can choose between the following options that apply when multiple dictionaries are searched using TEA:

  • Search all dictionaries independently – all the dictionaries are searched and all the results for each is shown (even if there are duplicates)
  • Search all dictionaries and suppress duplicated results – all the dictionaries are searched, but duplicates are suppressed (this is the default setting)
  • Search active dictionary – only the dictionary which has its list header highlighted is searched

Definition Dictionaries

Click on Change... to configure the Dictionary List that will be searched when the Edit|Lookup Command is done. TEA automatically searches the open dictionaries in the main window – this setting can be used to specify any additional dictionaries that you want to search.

The Dictionary List Dialog opens, allowing you to add and remove dictionaries in the list.

Once you have done this you can recall previously used dictionaries by clicking on the down arrow to the left of the Change… button and selecting the required entry.
