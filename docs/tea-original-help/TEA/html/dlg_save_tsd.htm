Save TEA&Sympathy Dictionary Dialog

Use the Save TEA&Sympathy Dictionary dialog to add a description to a TEA&Sympathy Dictionary File created from a list of results.

The description is shown in the box to the right of the Description label. By default, it is based on the name of the original dictionary and the pattern used to create the list of results being saved. You can update the description for the new dictionary by changing the text in this box.

Click on the Save button to create the new dictionary. Messages are shown in the box at the bottom of the dialog to indicate that rebuilding is in progress. If there is an error, it will also be shown in this box. If the dictionary is created successfully, the Save TEA&Sympathy Dictionary dialog closes automatically.
