Choice Lists

To match words where one or more letters must come from a restricted set, use a Choice List. Square brackets surrounding the set of letters indicate that any one of the letters can be chosen. For example, [aeiou] indicates that any vowel is allowed.

If a caret (^) appears at the start of the choice list, it indicates that any letter except the indicated set is allowed. For example, [^aeiou] indicates that any consonant is allowed.

Choice lists can be used in either the template part of a pattern or the anagram part of a pattern: in a template, the chosen letter must appear in sequence where the choice list appears; in an anagram, the chosen letter can be positioned anywhere in the word.

Macros are often used to abbreviate choice lists to a single character: by default, the @ macro expands to [aeiou] and the # macro expands to [^aeiou].

Examples

[xyz]..[abc] shows all the four-letter words
starting with X, Y or Z and
ending with A, B or C		 
yoga
zeta
zinc
;str[aeiou] finds words containing S, T, R
and any vowel		 
arts
erst
rest
rust
sort
star
stir
tsar
l[^aeiou]... finds five-letter words starting
with L and a consonant		 
llama
llano
lying
lymph
lynch
lyric
