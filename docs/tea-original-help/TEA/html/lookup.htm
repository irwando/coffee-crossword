Lookup

If you want to look up the Dictionary Definitions for an entry in a results list, first select the entry (see Selection). Then use the Edit|Lookup Command. (for which Ctrl+Enter is the keyboard shortcut). The definitions will appear in a TEA Definition Window.

You can open a definition in a new TEA Definition Window by using the same shortcut, but with the Shift key held down. This is equivalent to the Edit|Lookup in New Window Command.

You can also use the Edit|External Lookup Commands to search for an entry using another Windows application, e.g. an Internet search engine or a supported external Electronic Dictionary application.

Double-clicking on an entry is a quick way to see its definitions. Use the Options Result Tab to control whether double-click does a Lookup (i.e. the Edit|Lookup Command) or an External Lookup (i.e. the default Edit|External Lookup Command).
