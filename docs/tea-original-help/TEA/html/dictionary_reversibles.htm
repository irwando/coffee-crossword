Reversibles Dictionary Licensed Version Only

The Reversibles Dictionary includes English words and phrases that are another word (or phrase) in reverse.

This dictionary can be used to solve puzzles where the answer is known to have such a constraint. Because the Reversibles dictionary has such a specialized purpose, it is not opened by default, but can be opened using the File|Open Command.

Each entry includes an Annotation that shows its reversal. For example:

Entry Annotation
gateman name tag
Oprah Harpo
