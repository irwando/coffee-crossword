Conventions Used in Help

Terms used in the help are identified with dashed underlining, for example Anagram. Click on the term to see the definition in a pop-up window. The Glossary is an alphabetical list of all the terms.

Cross-references to other sections in the help are identified with solid underlining, for example Overview of Patterns. Click on a cross-reference to switch to that section. You can use the Back button to return to the original topic.

Many sections are grouped into sequences. You can use the << and >> buttons in the help application to move through the sections in the logical order.

The names of menu commands include | characters to separate the name of the menu from the name of the command. For example, File|Open means the Open command on the File menu.

Labels that appear in TEA dialogs are shown in bold face, for example Sort alphabetically. If you click on the Help button in a dialog to get context-specific help, you can correlate the bold text labels in the help with the different parts of the dialog.
