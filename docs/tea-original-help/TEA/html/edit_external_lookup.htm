Edit|External Lookup Commands

Use the Edit|External Lookup commands to search for the selected text using another Windows application, e.g. an Internet search engine or a supported external Electronic Dictionary application.

The names of these commands are based on the names specified in the Lookup Methods Dialog and indicate which external resource will be searched.

The Ctrl+Alt+Enter shortcut searches using the default method (the one with a filled circle in the menu). The default method is selected by placing a black check mark against it in the Lookup Methods Dialog).

If you are editing the Pattern, TEA will lookup the text you have selected (or the complete pattern if there is no selection). If you have selected an entry in a results list, TEA will lookup the word you have selected.

Note that most recent Electronic Dictionary applications support lookup from third-party software using the standard Edit|Copy Command: the Edit|External Lookup Commands have no effect with these applications. Instead, you need to press the Electronic Dictionary application's hot key, floating menu or the like; the Electronic Dictionary application then does a clipboard copy in TEA to find out which word is being looked up.
