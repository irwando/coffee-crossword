Letter Variables

Letter Variables are used to find words where the letters at certain positions are known to be the same. You can use any of the digits 0 to 9 to signify a letter variable in a pattern. If you check the Exclusive letter variables setting on the Options Pattern Tab, TEA ensures that each variable corresponds to a different letter.

Letter variables do not have to be used on their own in patterns: you can use them in combination with Match-All Characters, Wildcards and Choice Lists. They can appear in both the Template and Anagram part of a TEA pattern.

Letter variables have a number of uses illustrated in the examples below: they can help solve Cryptograms by identifying words that have a number of repeating letters. They can identify words with unusual properties such as Palindromes, Isograms and Tautonyms.
