Anagram Balances

When you match a pattern that includes an Anagram part, TEA can add letters that weren’t specified in the anagram or omit letters that were specified. The information about the additions and omissions is only displayed after the matched word if Show anagram balances is checked in the Options Result Tab.

The unspecified letters that are added appear after a plus sign (in this example a D is added):

;eiknrr. finds words using the letters EIKNRR
and one unspecified letter		 
drinker +D

The specified letters that are omitted appear after a minus sign:

.....;defijnx finds five-letter words from the selection
DEFIJNX		 
fiend –JX
fixed –JN
index -FJ

There can be both additions and omissions with some patterns:

......*;bdeeio. finds six-letter words from the selection
BDEEIO and one unspecified letter		 
beside -O+S
bodice -E+C
edible -O+L

The above examples show how anagram balances can be used when playing Scrabble®: the letters after the minus show what’s left over after a play and the letters after the plus show how any blank(s) have been used.

Anagram balances are shown using the same special font as for Annotations. This font is changed using the Results Annotation Font setting in the Options Appearance Tab.
