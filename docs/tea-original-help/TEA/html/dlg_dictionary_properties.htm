Dictionary Properties Dialog

Use the Dictionary Properties dialog to view information about a Dictionary shown in the TEA Main Window. To open this dialog, use the File|Properties Command, or double-click on a dictionary header.

The following information is shown about the selected dictionary:

Location
Shows where the TEA&Sympathy Dictionary File is located on your computer.
Date
Shows when the TEA&Sympathy Dictionary File was built.
Description
Shows the description added to the dictionary when it was created using the TEA&Sympathy Dictionary Builder application.
Headwords
Shows the total number of Headwords in the dictionary.
Definitions
Shows the number of Dictionary Definitions.
Shortest
Shows the shortest length of word in the dictionary.
Longest
Shows the longest length of word in the dictionary.
