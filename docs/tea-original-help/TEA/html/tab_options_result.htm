Options Result Tab

Use the Options Result Tab to change the way results lists are treated. To access these settings, use the Tools|Options Command and click on the Result tab.

Click on Save as defaults to use the settings on the tab as the defaults when TEA starts up.

Show dictionary annotations

Check this option to see Annotations in the lists of results.

Show anagram balances

Check this option to see Anagram Balances in the lists of results.

Filtering

Entries in word lists are classified into four different types for the purposes of filtering:

Entry type Example
Words containing only lower case letters 
tail
Words containing some upper case letters 
Charlie
Words containing punctuation 
tail-end
Multi-word phrases 
tail-end Charlie

Each of these types can be suppressed or shown by checking the corresponding option.

Double-click action

You can choose which type of lookup is done when you double-click on a result:

  • Lookup – double-click invokes the Edit|Lookup Command
  • External Lookup – double-click invokes the default Edit|External Lookup Command
